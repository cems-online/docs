# 6. Set Status Parameter?

Cara set status Parameter :
\>klik setting 
\> chimney 
\> pilih chimney 
\> pilih parameter yang ingin di set status nya 
\> klik kolom status akan muncul pilihan status
\> pilih status yang di inginkan
\> klik save
